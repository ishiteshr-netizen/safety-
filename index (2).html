⚙ Cloud Sync (Firebase Realtime Database)

Paste your Firebase web config JSON (with databaseURL) to enable Realtime Database sync & FCM push. Leave empty for offline-only. Sync is fully delta-based: indexed child_added / child_changed / child_removed listeners, compact records (~1 KB), no root reads, echo-suppression, archive for old records.

Paste these rules in Firebase Console → Realtime Database → Rules:
{ "rules": { ".read": true, ".write": true, "permits": { "active": { ".indexOn": ["u","st","n"] }, "archive": { "$y": { ".indexOn": ["u"] } } } } }
